### Decisions Made
- Bond pricing: FRED daily yield + standard semi-annual PV formula (`bondPricer.ts`)
- Stock/ETF bid/ask: `lastPrice × 0.0005` spread (0.05% — realistic large-cap spread)
- Bond bid/ask: fixed `±0.125` (standard 1/8 point)
- WS auth: JWT verified once per connection on AUTH frame; no per-message auth
- Price history: rolling in-memory array (last 30 entries); used by robo advisor momentum
- MDS does NOT issue JWTs — it only verifies them using shared `JWT_SECRET`
- Port 3001

### Decisions Made
- `client_order.token` changed from `INTEGER` to `TEXT` (JWT is a string)
- `client_preferences.accept_advisor` changed from `TEXT` to `BOOLEAN`
- `client_order.order_type` added: `VARCHAR(10) CHECK (order_type IN ('MARKET','LIMIT'))`
- `instrument` table added: `coupon_rate NUMERIC(6,4)`, `maturity_date DATE` for bonds
- All monetary fields: `NUMERIC(19,4)`
- All timestamps: `TIMESTAMPTZ`
- Hibernate `ddl-auto=validate` — Flyway owns schema, Hibernate never creates/alters
- `client_id` generated by Spring Boot as UUID on registration

- Spring Data JPA replaces MyBatis
- BCrypt cost factor 12
- Initial balance configurable via `app.initial-balance` property (default 10000)
- CORS allows `localhost:4200` only (no mid-tier)
- Fee: 0.1% both sides (BUY × 1.001, SELL × 0.999)
- Order types: MARKET (requires NYSE open) and LIMIT (IOC semantics)
